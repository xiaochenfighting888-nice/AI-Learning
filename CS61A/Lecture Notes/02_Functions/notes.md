# 函数—Functions

## 内置函数—Built-in-function

> 内置名称来自Python的built-in namespace，不需要`import`就可以直接使用。查找名字时，Python会先查局部作用域，再查全局作用域，最后查内置作用域。
>
> 库函数需要import导入，才能使用。

## 赋值—Assignment

> 把名字绑定到一个对象上。
>
> 赋值语句可以定义自己的名字，左边是名字，右边是表达式。
>
> 一条赋值语句可以同时绑定多个名字。执行时，先求值右边所有表达式，再进行左边名字的绑定。
>
> 赋值语句可以用来给函数起名字。

## 环境图—Environment Diagrams

> 用来可视化Python程序执行时变量绑定（name → object）和函数调用栈（frame）的示意图，这样就理解程序的执行过程。
>
> Frame（帧 / 框架）——存放名字到值的绑定
>
> ​	Global Frame（全局帧）——程序启动时创建
>
> ​	Local Frame（局部帧）——每次函数调用时新建
>
> Binding（绑定）——名字（name）指向值或对象

## 定义函数—Defining Functions

> 抽象，就是把复杂的事物赋予一个名称，将其视作一个整体来处理，而不必纠结于其内部的所有细节。
>
> 函数定义就是一种更强大的抽象方式，它将名字绑定在不仅仅是一个简单的值，而是整个表达式、语句或一系列语句上。其真正复杂的东西被赋予一个名字，并看作成一个整体，不用担心其细节。

## Print和None

> None表示函数“显式或隐式地返回了空值”。
>
> 没有明确返回值的函数将会返回None。
>
> 在Python交互式解释器中，如果表达式的值是`None`，解释器通常不会将它打印出来。

![局部截取_20260704_095214](../../images/局部截取_20260704_095214.png)

> Python中所有函数都有返回值。
>
> 纯函数只通过返回值产生结果，不产生副作用。
>
> 非纯函数除了返回值，还会产生“副作用”。函数不写`return`时，默认返回`None`，而交互式解释器里 `None`通常不显示。

# 问题

```python
from math import pi

def main():
    radius = 10
    area = pi * radius * radius
    print(area)
    radius = 20
    print(max(1, 3))

    f = max
    print(f(2, -1))
    max = 7
    print(f(1, max))

if __name__ == "__main__":
    main()
```

疑问：

​	为什么程序是一条条执行的，但在 `print(max(1, 3))` 就报错？

```text
UnboundLocalError: cannot access local variable 'max' where it is not associated with a value
```

解答：

​	在函数作用域里，Python会在执行函数体前先确定哪些名字是局部变量。只要函数体中出现了对某个名字的赋值语句，Python就会把这个名字视为局部变量。

所以在 `main` 里面，因为后面有：

```python
max = 7
```

Python会认为`max`是`main`的局部变量。于是前面的：

```python
print(max(1, 3))
```

不会去找内置函数`max`，而是试图访问局部变量`max`。但此时它还没有被赋值，所以报错。
